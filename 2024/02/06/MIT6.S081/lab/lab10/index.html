修改宏定义，然后多级读即可

Symbolic links

区别软链接与硬链接：

  • 软链接：它是一个特殊的文件（new file type (T_SYMLINK) ），文件的内容是一个路径，这个路径指向被链接的文件
  • 硬链接：它是一个目录项，这个目录项指向了目标文件

name是目录项的名字，inum则是它指向的inode块号，那么就是通过inum指向目标文件。

创建硬连接（ln t u）后，可以看到当前目录下多了一个u，查看它的内容与t一模一样。

创建软连接（ln -s t u）后，当前目录下多了一个u->t，它的大小为1（因为其内容为t的路径）。软链接是另外一种文件（u与t是不同的文件，inode也不同）。
